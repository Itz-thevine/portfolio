Cars NG

  • Category: Web development
  • Client: Cars NG
  • Project date: 10/3/2023
  • Project URL: cars.ng

Role

I've utilized Laravel Mix for bundling assets. Key dependencies include Axios for handling HTTP requests, Bootstrap for responsive UI components, and jQuery for DOM manipulation. Additionally, I've integrated browser-sync for live reloading during development and cross-env for setting environment variables. Lodash provides utility functions, while Popper.js handles tooltip and popover positioning. For CSS preprocessing, I've used Sass along with resolve-url-loader for resolving relative URLs. This ensures efficient development and a visually appealing, responsive user interface for the car sales app.
